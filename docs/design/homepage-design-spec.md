# AI Relay 首页设计规范

> 版本: v1.0 · 设计师: 像素姐 · 日期: 2026-05-24

## 1. 设计理念

将 AI Relay 从"技术文档页"升级为"开源项目官网"，采用深色赛博朋克风格，突出科技感和专业感。

**设计关键词**：科技感 · 极简 · 专业 · 可信赖

## 2. 色彩系统

### 主色调
- **背景色**：`#0a0a0f`（深空黑）
- **表面色**：`#111118`（卡片背景）
- **边框色**：`#1e1e2e`（分隔线/边框）

### 强调色
- **主强调**：`#00f2fe` → `#8b5cf6`（青紫渐变，用于 Logo、按钮、重点元素）
- **次强调**：`#3b82f6`（蓝色，用于链接、交互元素）
- **成功色**：`#10b981`（绿色，用于健康状态）
- **警告色**：`#f59e0b`（黄色，用于警告状态）

### 文字色
- **主文字**：`#e2e8f0`（明亮但不刺眼）
- **次文字**：`#94a3b8`（说明性文字）
- **弱文字**：`#64748b`（辅助信息）

## 3. 字体系统

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Noto Sans SC", sans-serif;
```

### 字号层级
- **H1（Hero 标题）**：3.5rem / 56px
- **H2（Section 标题）**：2rem / 32px
- **H3（卡片标题）**：1.25rem / 20px
- **正文**：1rem / 16px
- **小字**：0.875rem / 14px
- **代码**：0.85rem / 13.6px（等宽字体）

## 4. 间距系统

采用 8px 基准网格：
- **Section 间距**：120px（垂直）
- **卡片内边距**：32px
- **元素间距**：16px / 24px
- **内容最大宽度**：1200px

## 5. 组件规范

### 5.1 导航栏
- 固定顶部，半透明背景 `rgba(10,10,15,0.8)` + backdrop-blur
- 高度：64px
- 内容：Logo + 项目名 | 导航链接 | 语言切换
- 滚动时添加底边框 `1px solid #1e1e2e`

### 5.2 Hero 区域
- 全屏高度（100vh），垂直居中
- Logo 放大至 64px，带发光动画
- 标题下方添加渐变光晕效果
- CTA 按钮组：主按钮（渐变填充）+ 次按钮（边框）
- 背景：网格线 + 浮动光点动画

### 5.3 特性卡片
- 网格布局：桌面 3 列，平板 2 列，手机 1 列
- 卡片：圆角 16px，背景 `#111118`，边框 `1px solid #1e1e2e`
- Hover：边框变为渐变色，微微上移
- 图标：48px 圆形背景 + emoji
- P0 特性：默认展示
- P1 特性：可展开/折叠

### 5.4 代码块
- 背景：`#0d0d14`
- 圆角：12px
- 左侧：4px 渐变边框
- 顶部：语言标签 + 复制按钮
- 语法高亮：自定义深色主题

### 5.5 步骤指示器
- 水平时间线布局
- 步骤编号：圆形渐变背景
- 连接线：虚线，渐变色
- 当前步骤：高亮状态

### 5.6 按钮
- **主按钮**：渐变背景（`#00f2fe` → `#8b5cf6`），白色文字，圆角 8px
- **次按钮**：透明背景，渐变边框，渐变文字
- **幽灵按钮**：透明背景，弱边框，次文字色
- 所有按钮 Hover 时添加发光效果

## 7. 动画规范

### 7.1 入场动画
- **Hero 元素**：淡入 + 上移，延迟 0.1s 递增
- **特性卡片**：淡入 + 缩放，滚动触发
- **代码块**：逐行打字机效果（可选）

### 7.2 交互动画
- **按钮 Hover**：发光效果，持续 0.3s
- **卡片 Hover**：边框渐变 + 上移 4px，持续 0.2s
- **语言切换**：淡入淡出，持续 0.15s

### 7.3 背景动画
- **网格线**：缓慢移动的细线
- **光点**：随机浮动的小圆点
- **Logo 发光**：脉冲呼吸效果

## 8. 响应式断点

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 桌面 | ≥1024px | 完整布局 |
| 平板 | 768px-1023px | 2 列卡片，缩小间距 |
| 手机 | <768px | 单列，汉堡菜单，垂直堆叠 |

## 9. 语言切换设计

- **位置**：导航栏右侧
- **样式**：🌐 图标 + "中/EN" 文字
- **交互**：点击切换，页面内容淡入淡出
- **状态**：当前语言高亮显示

## 10. 可访问性

- 所有交互元素支持键盘导航
- 颜色对比度符合 WCAG AA 标准
- 图标配有文字说明
- 代码块支持屏幕阅读器

## 11. 性能优化

- 图标使用 SVG 内联
- 动画使用 CSS transform/opacity
- 背景动画使用 requestAnimationFrame
- 图片懒加载
